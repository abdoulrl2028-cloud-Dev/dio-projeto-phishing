# dio-projeto-phishing
 Este projeto demonstra, de forma **100% segura e ética**, como funciona um ataque de *phishing* utilizando o Kali Linux e a ferramenta **SET Toolkit** (*Social Engineering Toolkit*).   Nenhum serviço real (como Facebook, Gmail ou Instagram) é clonado ou atacado — toda a simulação é feita em **ambiente controlado**, como recomenda a DIO.
