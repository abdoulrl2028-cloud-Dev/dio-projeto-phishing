# Phishing - Simulação e Educação em Segurança

## Descrição do Projeto

Este projeto é uma simulação educacional de ataques de phishing em ambiente controlado e seguro. O objetivo é demonstrar técnicas de segurança ofensiva para fins educacionais e de conscientização.

## ⚠️ Aviso Legal

Este projeto é **apenas para fins educacionais** e deve ser utilizado em ambientes de teste controlados. O uso não autorizado de técnicas de phishing é **ilegal**. Consulte a seção `legal/aviso-legal.md` para mais informações.

## Como Usar

1. **Ambiente Seguro**: Execute este projeto apenas em máquinas virtuais ou ambientes isolados
2. **Consentimento**: Todos os participantes devem consentir explicitamente
3. **Educação**: Use como ferramenta de aprendizado sobre segurança da informação

## Requisitos

- Python (para servir as páginas)
- Navegador web
- Ambiente virtualizado recomendado

## Segurança

- Nunca use as técnicas aprendidas sem autorização explícita
- Este projeto é para conscientização e educação
- Todos os dados capturados devem ser imediatamente destruídos
